# HORP Allergy Filtering System

## Overview
A structured, rule-based menu filtering engine designed to streamline allergy and dietary communication in restaurant service operations.

## Problem
Restaurant service often slows down when handling multiple allergy and dietary restrictions...

## Solution
The system structures menu items into:
- Modular components
- Allergen tags
- Ingredient flags
- Cross-contact indicators
- Modification rules

Outputs include:
- Safe dishes
- Restricted dishes
- Modifiable dishes

## Tech Stack
- Node.js
- Vanilla JavaScript
- JSON structured data modeling
- Rule-based filtering engine
